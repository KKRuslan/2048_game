# 2048 Game

A fully functional, logic-driven web clone of the popular sliding tile puzzle game **2048**. Built using modern Object-Oriented JavaScript, standard semantic markup, and responsive styles, the project focuses on elegant algorithmic solutions for grid manipulation and state transition handling.

## Experience the live website:
[DEMO LINK](https://kkruslan.github.io/2048_game/)

## Technologies Used

### Core
* **HTML5** — Semantic markup structure and layout components.
* **SCSS** — Modular styling with nested architecture and variables.
* **JavaScript (ES6+)** — OOP class architecture handling core game algorithms.

### UI/UX
* **BEM Methodology** — Dynamic CSS class configuration for incremental tile values.
* **Matrix Rendering** — Synchronization of standard DOM lists with a $4 \times 4$ data grid.
* **Responsive Layout** — Flexbox alignment ensuring adaptive scaling across screens.

### Development & Deployment
* **Parcel (v2.12.0)** — Frontend asset bundling and compilation.
* **ESLint (v8.57.0)** — JavaScript code quality verification.
* **Stylelint (v16.7.0)** — SCSS syntax and style quality tracking.
* **GitHub Pages** — Automated project hosting and continuous static deployment.

## Getting Started
Follow these instructions to set up and run the project locally on your machine:

1. Clone the repository:
```bash
git clone https://github.com/kkruslan/2048_game.git
cd 2048_game
```
2. Install the necessary dependencies:
```bash
npm install
```
3. Run the project locally in development mode:
```bash
npm start
```

This will start a local development server, and you can view the website in your browser.
